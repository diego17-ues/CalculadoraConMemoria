# CalculadoraConMemoria
Calculadora con persistencia hecha en lenguaje Java y HTML, así como con persistencia haciendo uso PostgreSQL como gestor de bases de datos

Esta aplicacion web, ofrece una calculadora capaz de realizar operaciones básicas de forma eficiente; así como llevar un registro de las operaciones realizadas por el usuario.

Para ingresar a la aplicación, no se necesita ningun login; simplemente la calculadora se carga y puede disponer de ella.
Esta aplicación fue realizada con fines educativos, como parte del curso de Git y Github de Platzi para practicar el uso de la plataforma de control de versiones remotas GitHub.
